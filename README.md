## Notes

- Clone the repo from git hub.
- Run "npm init -y".
- Run "npm i express dotenv mongoose ".
- Run "npm i -D jest @code-fellows/supergoose".

**.env**
- MONGODB_URI = 'mongodb://localhost:27017/lab4'

**test**
- run "npm test"

**run the app**
- Run "node index.js"
